Firefox Add-On

May 23rd, 2012

Languages: JavaScript and some RDF

The AMO Up Top Version Info add-on makes browsing Mozilla’s add-on site, frequently called AMO, a little easier.

By default, an AMO odd-on page lists browser version compatibility at the very bottom of the page. Once you find the browser version section you have to click on the label to make the versioning information visible. AMO Up Top Version Info adds the most relevant version information to the top of an add-on page.

Before Image



After Image

There is also a UserScript version (see below) of this add-on. The UserScript version is both simpler and more efficient. The UserScript was converted to an add-on to learn about Firefox add-ons.

I now have a UserScript and a restartless/bootstrapped add-on using JetPack/Add-On Builder (big and cumbersome). I’m in the middle of implementing a restartless version of the add-on without JetPack. That requires a greater understanding of the XUL Framework but produces a much smaller add-on. Lastly, if it ever reaches the top of my priority list, I’ll write an old-school version of the add-on.
